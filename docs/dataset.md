# Dataset (Future)

OpenQuiz AI aims to build and share open educational datasets for question generation research. This feature is planned for a future release.

## Planned Datasets

- **Extracted Text Corpus** – anonymised collection of publicly available educational texts with their extracted and cleaned text.
- **Generated Questions Dataset** – AI‑generated multiple‑choice, true/false, and essay questions aligned with learning materials.
- **Difficulty & Bloom’s Taxonomy Labels** – human‑ or AI‑annotated questions classified by cognitive level and difficulty.
- **Multi‑subject Benchmarks** – standardised test sets for evaluating question generation quality across domains.

## Current Status

No datasets are included in Version 1.0. The `datasets/` directory is a placeholder. Contributions of open‑license educational texts are welcome – please open an issue to discuss.

## Related Work

We plan to follow the principles of established NLP datasets (SQuAD, RACE, SciQ) while focusing on educational assessment and providing metadata useful for adaptive learning systems.
