# OpenQuiz AI Developer Documentation

Selamat datang di dokumentasi pengembang OpenQuiz AI. Dokumentasi ini mencakup arsitektur, API, dan panduan kontribusi.

## Struktur Dokumentasi

- **Arsitektur** – Alur review, sesi generasi, metadata, logging, mode developer, clipboard, dan prompt pipeline.
- **API** – Endpoint generasi pertanyaan dan sistem review.
- **Kontributor** – Panduan memulai, struktur proyek, dan cara menambahkan fitur.

## Memulai

Lihat [Panduan Kontributor](contributing/developer-guide.md) untuk menjalankan proyek secara lokal dan memahami struktur kode.

## Ikhtisar Arsitektur

OpenQuiz AI mengikuti arsitektur berlapis: Frontend React → API FastAPI → Layanan → Provider AI. Detail lebih lanjut dapat ditemukan di [Alur Review](architecture/review-flow.md).
