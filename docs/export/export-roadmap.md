```markdown
# Export Roadmap

## Saat Ini (v1.0)

- ✅ PDF Exporter (ReportLab)
- ✅ ExportSession (in-memory)
- ✅ ExportCapability system
- ✅ Formatter Layer (plain, markdown, rich)
- ✅ Export Preview

## Rencana Jangka Pendek

- DOCX Exporter (python-docx)
- Markdown Exporter (file .md)
- CSV Exporter (daftar pertanyaan)
- HTML Exporter (halaman web)

## Rencana Jangka Panjang

- ZIP Bundle (beberapa format sekaligus)
- Moodle XML, Google Forms, Canvas LMS
- Ekspor terenkripsi, tanda tangan digital
- Ekspor latar belakang (queue)
- Riwayat ekspor dengan database
- Ekspor batch (banyak dokumen)
- Ekspor langsung ke cloud storage

## Cara Berkontribusi

Lihat [Panduan Pengembang](../contributing/developer-guide.md) dan ikuti pola yang ada: buat pengekspor, daftarkan, dan tambahkan kapabilitas.
```
