# OpenQuiz AI Roadmap

This roadmap describes the long-term development plan of OpenQuiz AI.

The goal is to build a complete AI-powered assessment platform step by step.

---

# Version 1.0 — AI Question Generator (MVP)

Goal:

Create a working AI application that can generate questions from learning materials.

Features:

- Upload PDF
- Upload DOCX
- Upload PPTX
- Paste text
- Generate Multiple Choice Questions
- Generate Essay Questions
- Generate True/False Questions
- Generate Answer Key
- Export to PDF
- Clean Web Interface

Status:

✅ Planned

---

# Version 2.0 — Smart Question Generation

# Long-Term Vision

OpenQuiz AI aims to become an open-source ecosystem for AI-powered educational assessment.

Rather than focusing on a single AI model, the project is designed to support:

- Modular architecture
- Multiple AI providers
- Open datasets
- Research reproducibility
- Community collaboration

Every version builds upon the previous one, ensuring the platform remains scalable, maintainable, and useful for educators, students, researchers, and developers.
